## Running

**Clone Repo :**

```bash
git clone https://github.com/irwansah/KP_IRWANSAHRULSIDIK_UJIKETERAMPILAN
```

```bash
cd uji_keterampian
```

**Install all dependency :**

```bash
flutter pub get
```

**Run:**

```bash
flutter run
```
